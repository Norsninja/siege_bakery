## 0. The shape of the path

Three lanes run in parallel, because they consume different
resources:

- **THE BUILD LANE** — session work, sequential, one claim at a time.
- **THE FLEET LANE** — asset work dispatchable to Blender fleet
  agents (research/18 contract) while build sessions proceed.
- **THE DESIGN LANE** — the visionary's discussion time; sessions
  that produce rulings, not code. Each unlocks build work but none
  blocks the build lane's current head.

Standing law from the milestone: core/ and game/ ship untouched by
presentation work; the two exceptions on the path (the giant
collider's core capsule data, the species promotion) are named where
they occur.
